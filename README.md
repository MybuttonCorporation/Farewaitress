# Farewaitress Documentation
### Install With git
    
    git clone https://github.com/MybuttonCorporation/Farewaitress.git  
    npm install
    
### Run
    
    
    node index.js
    
    
## NOTE ##

__You should change the name of `config_template.json` in the `./config` folder to `config.json`.__
